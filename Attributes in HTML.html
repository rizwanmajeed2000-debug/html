Tables

Tables are used with the tags of table, also adding the border size, and then specifying columns, rows and data cells. For example,
Name Age City
Javeria 13 Lahore

Ordered Lists

In html, Ordered lists are used to write a list written in numbers accordingly. For example,
  1. Karachi
  2. Multan
  3. Faisalabad

Unordered Lists

Unordered Lists write the data with bullets. For example,
  • Karachi
  • Multan
  • Faisalabad
Descriptive Lists
In Descriptive lists, the data is described a bit more to spefify it, i.e.,
Coffee
A hot black drink to energize you.
Tea
A drink with leaves.
Milk
A nutritious drink.
